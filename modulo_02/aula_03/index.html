Exercício 1: Encontrar o Maior Número

Crie uma função chamada encontrarMaiorNumero que recebe um array de números como argumento e retorna o maior número presente nesse array.

Exemplo de uso:

console.log(encontrarMaiorNumero([10, 3, 7, 22, 19])); // Saída esperada: 22
console.log(encontrarMaiorNumero([5, 11, 2, 8])); // Saída esperada: 11
console.log(encontrarMaiorNumero([4, 4, 4, 4])); // Saída esperada: 4
