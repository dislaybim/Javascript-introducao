Manipulação de array:

As funções de manipulação de arrays em JavaScript são extremamente úteis e poderosas para realizar operações como adicionar, remover, modificar e percorrer elementos em um array. Elas permitem manipular os dados de forma eficiente e expressiva. Abaixo, vou apresentar algumas das funções mais comuns para manipulação de arrays:

1. `push()`
A função `push()` adiciona um ou mais elementos ao final de um array e retorna o novo comprimento do array.
Exemplo:
javascript
let frutas = ['maçã', 'banana'];
frutas.push('laranja', 'uva');
console.log(frutas); // Saída: ['maçã', 'banana', 'laranja', 'uva']

2. `pop()`
A função `pop()` remove o último elemento de um array e retorna esse elemento removido.
Exemplo:
javascript
let frutas = ['maçã', 'banana', 'laranja'];
let ultimaFruta = frutas.pop();
console.log(ultimaFruta); // Saída: 'laranja'
console.log(frutas); // Saída: ['maçã', 'banana']

3. `shift()`
A função `shift()` remove o primeiro elemento de um array e retorna esse elemento removido, atualizando os índices dos outros elementos.
Exemplo:
javascript
let frutas = ['maçã', 'banana', 'laranja'];
let primeiraFruta = frutas.shift();
console.log(primeiraFruta); // Saída: 'maçã'
console.log(frutas); // Saída: ['banana', 'laranja']

4. `unshift()`
A função `unshift()` adiciona um ou mais elementos no início de um array e retorna o novo comprimento do array.
Exemplo:
javascript
let frutas = ['banana', 'laranja'];
frutas.unshift('maçã', 'uva');
console.log(frutas); // Saída: ['maçã', 'uva', 'banana', 'laranja']

5. `splice()`
A função `splice()` permite adicionar, remover ou substituir elementos em qualquer posição de um array.
Exemplos:
javascript
// Adicionar elementos
let frutas = ['maçã', 'laranja', 'uva'];
frutas.splice(1, 0, 'banana'); // Adiciona 'banana' na posição 1
console.log(frutas); // Saída: ['maçã', 'banana', 'laranja', 'uva']
// Remover elementos
frutas.splice(2, 1); // Remove 1 elemento a partir da posição 2
console.log(frutas); // Saída: ['maçã', 'banana', 'uva']
// Substituir elementos
frutas.splice(1, 1, 'melancia'); // Substitui 'banana' por 'melancia' na posição 1
console.log(frutas); // Saída: ['maçã', 'melancia', 'uva']

6. `concat()`
A função `concat()` combina dois ou mais arrays, retornando um novo array sem modificar os arrays originais.
Exemplo:
javascript
let frutas = ['maçã', 'banana'];
let outrasFrutas = ['laranja', 'uva'];
let todasFrutas = frutas.concat(outrasFrutas);
console.log(todasFrutas); // Saída: ['maçã', 'banana', 'laranja', 'uva']

7. `map()`
A função `map()` cria um novo array com os resultados de uma função aplicada a cada elemento do array original.
Exemplo:
javascript
let numeros = [1, 2, 3];
let dobrados = numeros.map(num => num * 2);
console.log(dobrados); // Saída: [2, 4, 6]

8. `filter()`
A função `filter()` cria um novo array com todos os elementos que passam por um teste (fornecido como uma função).
Exemplo:
javascript
let numeros = [1, 2, 3, 4, 5];
let pares = numeros.filter(num => num % 2 === 0);
console.log(pares); // Saída: [2, 4]

9. `reduce()`
A função `reduce()` executa uma função redutora (fornecida) para cada elemento do array, resultando em um único valor.
Exemplo:
javascript
let numeros = [1, 2, 3, 4];
let soma = numeros.reduce((total, num) => total + num, 0);
console.log(soma); // Saída: 10 (soma de todos os números)

10. `forEach()`
A função `forEach()` executa uma função para cada elemento do array.
Exemplo:
javascript
let frutas = ['maçã', 'banana', 'laranja'];
frutas.forEach(fruta => {
console.log(fruta);
});
// Saída:
// maçã
// banana
// laranja

Essas são apenas algumas das muitas funções úteis para manipulação de arrays em JavaScript. Elas são fundamentais para trabalhar com dados em JavaScript de forma eficiente e expressiva.
